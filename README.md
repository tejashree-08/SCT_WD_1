# 🚀 SkillBridge – Responsive Landing Page

SkillBridge is a modern and fully responsive landing page designed to connect students with internships, mentors, career roadmaps, and placement opportunities.

This project was developed as part of the **SkillCraft Technology Web Development Internship**.

---

## 🌟 Features

### ✅ Interactive Navigation Menu
- Fixed navigation bar
- Visible on all sections/pages
- Changes appearance on scroll
- Hover animations on menu items
- Active section highlighting
- Smooth scrolling navigation

### ✅ Responsive Design
- Desktop friendly
- Tablet optimized
- Mobile responsive
- Hamburger menu for smaller screens

### ✅ Modern UI/UX
- Glassmorphism design
- Gradient accents
- Interactive cards
- Smooth animations
- Professional SaaS-inspired layout

### ✅ Sections Included
- Hero Section
- Features
- Internship Categories
- Career Roadmaps
- Mentors
- Success Stories
- How It Works
- Statistics
- FAQ
- Contact Form
- Footer

### ✅ Interactive Components
- Animated counters
- Testimonial carousel
- FAQ accordion
- Form validation
- Scroll progress indicator
- Reveal-on-scroll animations
